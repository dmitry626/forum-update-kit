# Шаблон подготовки к апдейту

Подготовка — промежуточный документ. Здесь ты собираешь сырой материал и превращаешь его в 5%. Потом из подготовки пишешь финальный апдейт.

Копируй этот файл, заполняй в три фазы.

---

## Фаза 1. Досье периода

Хронологический список фактов за период **с даты последнего апдейта по сегодня**. По 4 сферам.

Факты — что случилось, а не что ты об этом думаешь. Если в дневнике или переписке есть прямая эмоциональная фраза — цитируй её, не пересказывай.

## Фаза 2. Зеркало — что могло бы быть в 5%

Для каждого блока досье: какие тезисы **могут** оказаться в 5%? Проверь по 7 маркерам значимости из [`methodology.md`](methodology.md):

1. Смена роли / идентичности
2. Угроза ключевой связи
3. Телесный след
4. Полярные эмоции
5. Впервые
6. Личная уязвимость
7. Внутренний сдвиг

Формат: тезис → маркер → чего хватает / не хватает для полноценного 5%-материала.

## Фаза 3. Материал для запроса

Самая важная часть подготовки. Цель — **сформулировать запрос к форуму**, который пройдёт три критерия: внутренний, уязвимый, свежий.

### 3.1. Карта прошлых запросов

Если у тебя есть история прошлых форум-апдейтов — выпиши запросы последнего года. Если нет — вспомни: какой вопрос ты уже задавал форуму за последнее время?

| Дата | Запрос |
|:---|:---|
| | |

**Повторяющиеся темы:**
- (Если одна и та же тема поднималась 2+ раза — назови её)

### 3.2. Противоречия за период

Самый сильный источник запроса — противоречие между тем, что ты **говорил / думал / обещал** и тем, что **делал** за период.

| Слово / убеждение | Действие / факт | Вопрос |
|:---|:---|:---|
| | | |
| | | |

Формат вопроса: «Я считаю X, и при этом делаю Y — что я на самом деле делаю?»

### 3.3. Детектор повторов

Если тема текущего запроса совпадает с повторяющейся темой из 3.1:
- **Явно назови:** «Я поднимаю эту тему N-й раз».
- **Что изменилось?** За прошедший период между встречами — что нового в твоём понимании? Какое действие ты предпринял после прошлого разбора?
- **Что нового ты ждёшь от группы?** Если ждёшь того же самого — тема, вероятно, не созрела. Нужен рефрейминг.

### 3.4. Варианты запроса

2-3 черновика формулировки. Не финальная — просто разные углы.

- **V1:** ...
- **V2:** ...
- **V3:** ...

Из них выбираешь один — или собираешь финальный из кусочков.

### 3.5. Вопросы для самопроверки

Честно ответь себе:

- **Чего я сейчас избегаю?**
- **Какое самое важное решение мне предстоит принять?**
- **Чем я менее всего удовлетворён?**
- **В чём самый большой страх прямо сейчас?**
- **Чему я сопротивляюсь?**
- **В чём я не уверен?**

Один из этих ответов — вероятный кандидат в запрос.
